Chrome中的新变化

新版Chrome应用商店以及安全在线连接
CHROME 网上应用店

浏览新类别和个性化推荐内容，轻松找到契合您需求的主题和扩展程序，从而定制您的专属浏览器。

  • 1.打开 Chrome 菜单图标。
  • 2.依次选择扩展程序和访问 Chrome 应用商店。
  • 3.浏览各种扩展程序和主题，定制您的专属 Chrome。
CHROME 小窍门

安全连接，安心浏览

如果您通过不安全的连接访问某个网站，Chrome 可在连接该网站前向您发出警告。这可确保您在上网时更安全，尤其是使用咖啡馆和机场等场所的公共网络时。

  • 1.打开 Chrome 菜单，然后选择设置。
  • 2.点击隐私和安全，然后选择安全。
  • 3.在高级部分中，开启一律使用安全连接。
