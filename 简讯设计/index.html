Improvements for days

  • You can now log in with Twitter or Facebook now, in addition to GitHub or "regular style". You can also change your avatar by grabbing from any of those services.
  • You can edit Pen titles right where you see them in the Editor or Details View now.
  • Preprocessor add-ons are now easier to find and work with.
  • The console now supports Sass output and the clear() command.
  • We designed our own default avatar, rather than using the default Gravatar one.
  • JSX is now properly syntax highlighted.
  • We added five brand new syntax highlighting themes and Operator Mono as a new font choice.
